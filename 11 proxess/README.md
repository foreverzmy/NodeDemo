## 02.js：通过uncaughtException事假捕获异常

说明：在Node中，时间主循环里碰到的异常会导致Node进程退出。uncaughtException事件提供了一个极其暴力的方法来捕获这些异常。
uncaughtException是一个简单的智能处理程序，只能简答的把异常输出到标准输出，记录下这些错误。但是，因为它捕获的是一个不存在的函数触发事件，所以虽然Node程序不会退出，但是标准的执行流程会被打断。
